# API REST com Spring Boot ★
Este é um projeto de API REST desenvolvido em Java, utilizando o Spring Boot, e testada via Postman.

## Tecnologias Utilizadas: ♬
- Java 21;
- Spring Boot 3+;
- Maven;
- Spring Web;
- Spring DevTools;
- Postman (para testes de requisição);
- Eclipse IDE.
